Magic DXLink Configurator

Magic DXLink Configurator is a program that integrates unit discovery and telnet commands to ease configuration and management of AMX DXLink devices.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Magic DXLink Configurator.

Have you ever plugged in a new DXLink transmitter or receiver and wondered what IP address it received? Would it be great if you could send a VIDOUT_RES_REF of 1920x1080p, 60 to twenty DXLink receivers at once?

Have you been given a list of IP addresses and need to find out what their serial number, firmware version, and device configuration information?

The Magic DXLink Configurator is an internal software tool that we developed here at AMX Australia. It is a graphical front end to the telnet interface on DXLink devices. It allows you to do almost all telnet commands with ease, from a friendly graphical interface.

It includes the following features:

  • Detecting when a DXLink device receives a DHCP address.
  • Collect information on DXLink devices for an entire subnet.
  • Quickly configure multiple DXLink devices to connect to a master, check firmware version loaded and current configuration.
  • Use “send command” on a single device or multiple devices at once.

We have had a number of requests for it to be released to the public. So the program is being released open source, and you can modify the code to meet your needs.

If this sounds like something that you would be interested in, head over to our github page at http://github.com/AMXAUNZ/Magic-DXLink-Configurator/releases
